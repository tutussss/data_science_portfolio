When filtering by date range, Tableau also offers the options Starting date and Ending date.

When we choose Starting date, we set the earliest date in our date range, while the end date is set to the latest date available in the data source.

Let's view what we have selected in the following screenshot:

On the other hand, with the Ending date option, we set the latest date in the date range that we would like to filter by, while the starting date is set to the earliest available date in our data source.
