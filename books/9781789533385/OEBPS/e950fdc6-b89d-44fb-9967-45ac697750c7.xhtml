  1. Drag and drop Country from Dimension onto Detail in the Marks card.
  2. Drag and drop Points from Measures onto Color in the Marks card.
  3. Right-click on the SUM (Points) pill in Color, navigate to Measure (Sum), and from the drop-down menu, select Average.
  4. Create a new blank worksheet by clicking on the New Worksheet tab at the bottom of the workspace. Alternatively, in the main menu toolbar, navigate to Worksheet | New Worksheet from the drop-down menu.
  1. In the new worksheet, drag and drop Country from Dimensions onto Color in the Marks card. A dialog window may pop up, asking you if you would like to filter out some of the countries. Select Add all members:

  1. Drag and drop Province from Dimensions onto Detail in the Marks card.
  2. Drag and drop Price from Measures onto Size in the Marks card.
  3. Right-click on the SUM (Price) pill in Size, navigate to Measure (Sum), and from the drop-down menu, select Average.
  4. In the Marks card, click on the drop-down menu and change the mark type from Automatic to Circle:

  1. Navigate to Sheet 1. In the main menu toolbar, select Worksheet and in the drop-down menu, navigate to Actions...:

  1. In the Actions window that opens, click on the Add Action > button and select Filter...:

  1. In the Add Filter Action window, use the Name field to change the name of the filter from Filter1 to Filter per Country.
  2. Under Run action on, select Select.
  3. Set Target Sheets to Sheet 2.
  4. Click OK to exit the window:

  1. You will notice that our filter, Filter per Country, has now appeared in the list in the Actions window. Click OK to exit the Actions window.
  2. We have now set up the action filter. Click on any country in the map on Sheet 1 and observe the result on Sheet 2. Notice that the action filter has now also appeared in the Filters shelf.
