How to do it...

  1. Launch Tableau.
  2. In the main menu toolbar, select Help.
  3. In the drop-down menu, go to Settings and Performance, and, from the additional drop-down menu, select Start Performance Recording:

  1. In the main menu toolbar, click on File and select Open….
  2. Navigate to the location on your device where you've saved the dashboard, select it, and click on Open.
  3. Try playing around with the dashboard a bit—for example, filter different options.
  4. In the main menu toolbar, navigate to Help.
  1. In the drop-down menu, select Settings and Performance and in the additional drop-down menu, select Stop Performance Recording:

  1. Once you've stopped the performance recording, you may need to wait a couple of seconds for Tableau to generate the performance recording report. Tableau will automatically open a new read-only workbook, called PerformanceRecording.
