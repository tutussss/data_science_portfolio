For the production version, you should consider using Elastic Load Balancer (ELB) that will serve as an end point for the Tableau Server. In addition, you can upload SSL certificates to ELB and make your connection secure. Finally, you can use the ELB logging option and collect web logs.

You can run Tableau Server on Linux with a distributed environment. There is good example of it at Tableau documentation, which can be found here:

https://onlinehelp.tableau.com/current/server-linux/en-us/ts_aws_multiple_server.htm.
