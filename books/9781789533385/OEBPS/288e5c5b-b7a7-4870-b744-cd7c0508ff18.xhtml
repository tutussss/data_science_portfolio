How to do it…

  1. In the main menu toolbar, navigate to Map.
  2. From the drop-down menu, navigate to Geocoding | Import Custom Geocoding… as shown in the following screenshot:

  1. In the Import Custom Geocoding window, click on three dots (...) to open the directory where you saved Province_geocoding.csv:

  1. In the Choose Source Folder window that opens, navigate to the folder in which you have saved the Province_geocoding.csv file with custom geocoding data, select it, and click Select Folder.
  2. The path to the folder will now appear in the Import Custom Geocoding window. Click on Import, and wait while Tableau processes your request.
  3. When the custom geocoding is imported, right-click on the Province pill under Dimensions, and in the drop-down menu navigate to Geographical Role and choose Province-Serbia:

  1. We have now set up our custom geocoding for Province-Serbia, which is also indicated by a small globe with a paper symbol () that has appeared next to it. To use it in a map, just drag and drop Province from Dimensions into the workspace.
  2. Finally, add Population, by dropping it from Measures onto Size in the Marks card as shown in the following screenshot:
