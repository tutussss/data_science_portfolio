Filtering out cases in the database

If you aren't going to use all the cases from your database, the best thing you can do is apply a filter to it:

  1. Open any workbook that you want to improve, and go to the Data Source tab in the bottom-left corner:

  1. In the top-right corner, you can see the Filters section. Select Add:

  1. The list of all of the variables in the database will be presented.
  2. Chose the variable that contains the values you want to filter out:

  1. Select only those that you want to preserve (in our example, we want to keep only answers from female respondents), as shown in the following screenshot:
