Getting ready

In this recipe, will will download the last available Tableau Server for Linux and deploy it in our AWS account on top of EC2 Linux Instance.

We can use the same VPC that we used for the Windows recipe, or we can deploy new from scratch. In this recipe we will:

  • Launch EC2 Linux instance
  • Download Tableau Server for Linux distribution
  • Install Tableau Server and access it via SSH

Before you start, you also may refer to Everybody's Install Guide, which will give you a more detailed explanation behind the server installation at https://onlinehelp.tableau.com/current/guides/everybody-install-linux/en-us/everybody_admin_intro.htm.
