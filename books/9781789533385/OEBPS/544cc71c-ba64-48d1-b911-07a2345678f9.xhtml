Filtering out variables

In most cases, we don't need all of the variables that we have in our database. In order to make our workbook run smoothly, we can hide variables that we don't need:

  1. Open any workbook that you want to improve, and go to the Data Source tab in the bottom-left corner. Hover over the variable that you want to exclude until a black downward arrow appears.
  2. Click on it and select Hide:
