Building automation with Tabcmd is awesome, but it has one missing element—notifications. In the real world, I am using an open source email client for CLI: https://www.febooti.com/.

You can download this tool and try it. All you need is to insert a snippet with the Febooti command and it will deliver you an email or attachment with the file. End users love it!

Finally, you should consider using internal Windows Task Scheduler to schedule Tabcmd scripts (Batch files) that can be found here: https://en.wikipedia.org/wiki/Windows_Task_Scheduler.
