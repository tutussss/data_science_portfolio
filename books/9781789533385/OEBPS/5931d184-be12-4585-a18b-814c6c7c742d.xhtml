How it works...

In this recipe, we chose to select all of the records in our dataset that fall within a certain date range. We specified the start date and the end date, and Tableau filtered out all dates that fall out of this range.

Although date filters are specific for the data field type, they work just like any other filter. When we specified the range of dates, Tableau filtered out all of the rows in our data source that contain date values that are outside the specified range, and included only the rows with the date values that fall within the specified range in our view.
