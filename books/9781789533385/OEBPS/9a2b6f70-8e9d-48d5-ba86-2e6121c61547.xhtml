How to do it…

  1. Drag and drop Country from Dimensions into the Rows shelf.
  2. Drag and drop Number of Records from Measures into the Columns shelf.
  3. Drag and drop Taster Name from Dimensions onto Color in the Marks card.
  4. Right-click on the Number of Records pill in the Columns shelf and in the drop-down menu navigate to Quick Table Calculation | Percent of Total:

  1. Once again, right-click on the Number of Records pill in the Columns shelf and in the drop-down menu navigate to Compute Using | Taster Name:

  1. We can now see the percentage of reviews done by each of the tasters in each of the countries. However, the view looks pretty messy, and it's difficult to identify individual wine tasters, as shown in the following screenshot:

  1. Let's say we want to filter out a single taster, and see their share of reviews in each country. Drag and drop Taster Name from Dimensions into the Filters shelf.
  2. In the Filter [Taster Name] window, click on None to deselect all names. Then, select Anna Lee C. Iijima and click OK:

The view we got is not what we needed! See the following screenshot:

  1. Let's correct that. In the main menu toolbar, click on Analysis and select Create Calculated Field….
  2. In the calculated field editor window, rename the calculated field from Calculation1 to Filter by Wine Taster.
  3. In the formula space, write the following expression:
LOOKUP(ATTR([Taster Name]),0)

The preceding expression is shown in the following screenshot:

  1. Click OK to exit the window.
  2. Remove Taster Name from the Filter shelf and drag and drop the new calculated field, Filter by Wine Taster, into the Filter shelf instead.
  1. In the Filter [Filter by Wine Taster] window, select Anna Lee C. Iijima and click OK. The view now shows the percentage of records by this wine taster in each country:
