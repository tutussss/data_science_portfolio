Customizing the device layout

  1. Once you click the Add Desktop/Tablet/Phone Layout button, the new layout you added will appear in the Dashboard pane, alongside the default layout:

  1. Click on different layouts that you have added to the Dashboard pane to preview them.
  2. When you click on a device, such as the Tablet layout, you can set the size of the dashboard on the Tablet screen in the Dashboard pane, under Size – Tablet. Check the radio button in front of Fit all to fit the entire dashboard onto a tablet screen, or check Fit width to fit the dashboard by width, while setting the height manually in the Height box.
  1. Under Layout – Tablet, you can leave the default option selected, and you can also check Custom. If you check Custom, a menu that allows you to manipulate dashboard elements will appear:
