How it works...

In order to highlight the final value in the Sparkline, we create a calculation for the final value in step 3, put it on the Rows shelf in step 4, and, in step 5, make it a synchronized dual axis chart. Each of the line charts should appear independent so we can see changes for each category relative to itself and not to each other.
