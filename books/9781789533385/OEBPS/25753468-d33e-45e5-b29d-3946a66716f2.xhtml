Setting the range size

  1. In the Dashboard pane on the left-hand side, under Size, click on the drop-down menu.
  2. In the first drop-down menu that appears, switch from Fixed size to Range.
  3. Set the minimum screen size either by typing in the desired Width and Height in the appropriate boxes, or using the up and down arrows adjacent to the boxes.
  4. Set the maximum screen size using the same procedure.
